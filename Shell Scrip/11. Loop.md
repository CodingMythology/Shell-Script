# While

```
while [condition]
do 
    statement1
    statement2
    statement3
done
```
Example
```
n = 1;
while [ $n -le 10]
do 
    n = $(( n++))
done
```

# Until Loop
```
until [condition]
do 
    statement1
    statement2
    statement3
done
```

### Note here
if we use flag `-` we need `[]`

if we use sign `<` we use `(())`
