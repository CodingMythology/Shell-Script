# Type of Network

network一般是由reach和size构成的。
reach 有三种 LAN/MAN/WAN


# reach

### LAN (Local Area Network)
这种一般在比较小的区域中人们希望能够share文件

          |small boundary|
    |-------------------------|
    | |Computer| /——|printer| |
    |----|------/-------------|
    | |Sever|—————|computer|  |
    |-------------------------|
一般是用wireless或是cable

### MAN (Metropolics Area Network)
这种一般在城市中,建筑与建筑之间联系

          |City|
    |----|       |----|
    |    |       |    |
    |    | <---> |    |
    |    |       |    |
   |———————————————————————|
一般是用cable来链接

### WAN (widest Area Network)
这种就是城市与城市之间联系(<=30miles)

          |Country|
 |--------------------------|       
 |  |City| ————l———— |City| |
 |     |————|City|————|     |
 |--------------------------|
 一般用cable和基站（因为大于30mile的要用基站）
